# 📘 Agile Resources & Ceremony Guides

A collection of practical Agile guides, templates, and tips based on real‑world experience.  
These resources are adapted from my Medium articles and are designed to help Scrum teams improve clarity, predictability, and collaboration.

Whether you're a Scrum Master, Agile Coach, Product Owner, or developer working in a Scrum environment, this repository provides actionable tools you can use immediately.

---

## 🧭 Purpose of This Repository

Agile teams thrive when they have:

- Clear ceremonies  
- Strong facilitation  
- Transparent metrics  
- Consistent processes  
- Shared understanding  

This repository brings together the guides, templates, and insights I use with teams to help them deliver value more effectively.  
Each guide is a distilled, GitHub‑friendly version of a full Medium article, with a link back to the original for deeper reading.

**https://ericspink.medium.com/**
